# KeyShield AI

## Enterprise Behavioral Authentication & Analytics Platform

KeyShield AI is a full-stack behavioral biometric authentication system that verifies users based on their typing behavior using Machine Learning, Explainable AI (XAI), and Data Engineering.

### Tech Stack

- React
- FastAPI
- PostgreSQL
- Scikit-learn
- SHAP
- Python
- SQL
- Tailwind CSS

> 🚧 Project currently under active development.
